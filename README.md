# AI Network Threat Detector

One-command installation and usage!

## Quick Start
```bash
git clone https://github.com/nirvana58/ai-based-network-threat-detector.git
cd ai-network-threat-detector
./threat_detector.sh
```

That's it! The script handles everything.

Remember to install ollama to local system as to script to pull during analysis

## First Time Use

1. Run `./threat_detector.sh`
2. Wait for automatic setup
3. Select [1] to generate sample data
4. Select [2] to train model
5. Select [3] to analyze threats

## Requirements

- Linux/Mac
- Python 3.9+
- Ollama
